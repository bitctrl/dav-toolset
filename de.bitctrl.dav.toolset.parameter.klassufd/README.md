# Tool zur Parametrierung der Klassifizierung für Umfelddatensensoren

Gesetzt werden die in den Anforderungen des Segment DuA vorgebenenen Grenzwerte für Sensoren der Typen:

- Niederschlagsintensität
- Wasserfilmdicke und
- Sichtweite

Version: 1.0.0
